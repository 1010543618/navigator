    About PaleoDB Navigator

    The Application

    Built on the PaleoDB API, PaleoDB Navigator allows users to explore the Paleobiology Database through space, time, and taxonomy. Paleobiology Database data is used under a CC-BY 3.0 Creative Commons license. In addition to the Paleobiology Database, it utilizes data from GPlates under a GNU General Public License (GPL), version 2 license for plate and collection rotations, and Phylopic.org for taxonomic images under either a Public Domain or Creative Commons license. All code unique to the PaleoDB Navigator interface uses a CC0 1.0 Universal Public Domain Dedication, and the all the code for it can be found on insert link to Github repo here.


    Project Funding

    Current development of the PaleoDB database, API, website, and example applications is being carried out in the Department of Geoscience at the University of Wisconsin-Madison. It is supported by the UW-Madison Deptartment of Geoscience and the U.S. National Science Foundation EAR Geoinformatics grant 0949416.
